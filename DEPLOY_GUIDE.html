🚀 Netlify デプロイ手順

⚠️ 重要：このページに記載されている手順に従って、必要なファイルをダウンロードして Netlify にアップロードしてください。

📦 必要なファイル構成

プロジェクトフォルダ/ ├─ index.html (31.9 KB) ├─ dashboard.html (5.2 KB) ├─ analytics.html (6.3 KB) ├─ css/ │ ├─ style.css (11.5 KB) │ ├─ dashboard.css (6.0 KB) │ └─ analytics.css (2.4 KB) └─ js/ ├─ config.js (2.2 KB) ├─ main.js (18.2 KB) ├─ dashboard.js (18.9 KB) ├─ analytics.js (15.0 KB) └─ auth.js (1.9 KB) ← ★ この認証ファイルが重要！

📥 ステップ1：ファイルをダウンロード

方法A：個別にダウンロード（推奨）

  1. AI チャット画面の左側「ファイルエクスプローラー」を開く
  2. 以下のファイルを右クリック → ダウンロード：
    • index.html
    • dashboard.html
    • analytics.html
  3. css/ フォルダを右クリック → ダウンロード
  4. js/ フォルダを右クリック → ダウンロード

方法B：一括ダウンロード

  1. ファイルエクスプローラーの上部にある「ダウンロード」アイコンをクリック
  2. すべてのファイルが ZIP 形式でダウンロードされます
  3. ダウンロードした ZIP を解凍

📤 ステップ2：Netlify にアップロード

アップロード手順

  1. Netlify にログイン
  2. サイト stellular-profiterole-2ff0a2 を開く
  3. Deploys タブをクリック
  4. 右上の 「Trigger deploy」→「Clear cache and deploy site」 をクリック
  5. 画面に表示される 「Drag and drop your site output folder here」 エリアに、 以下のいずれかをドラッグ：
    • フォルダ全体（index.html、css/、js/ を含むフォルダ）
    • ZIP ファイル（解凍せずにそのまま）
  6. アップロード完了を待つ（1〜2分）
⚠️ 注意：必ず js/auth.js ファイルが含まれていることを確認してください。このファイルがないと認証機能が動作しません。

✅ ステップ3：動作確認

テスト手順

  1. デプロイ完了後、シークレットモード（Ctrl+Shift+N）でブラウザを開く
  2. https://stellular-profiterole-2ff0a2.netlify.app/index.html を開く
  3. 以下のダイアログが表示されることを確認：
    • ユーザーID：nlp-test
    • パスワード：nlp2026
  4. 正しく入力すると「ログインしました」と表示される
  5. 「キャンセル」または空白で OK を押すと「アクセスが拒否されました」と表示される

🔧 トラブルシューティング

認証ダイアログが表示されない場合

  1. F12 を押して開発者ツールを開く
  2. Console タブを確認
  3. 赤いエラーが表示されている場合：
    • auth.js:1 Failed to load resource → auth.js がアップロードされていません
    • 解決方法：js/ フォルダ内に auth.js が含まれていることを確認して再アップロード
  4. Network タブで auth.js が 200 (成功) になっているか確認

それでも動作しない場合

  1. Netlify で 「Clear cache and deploy site」 を実行
  2. ブラウザのキャッシュをクリア（Ctrl+Shift+Delete）
  3. シークレットモードで再度テスト
  4. 直接 https://stellular-profiterole-2ff0a2.netlify.app/js/auth.js を開いて、 コードが表示されるか確認

📞 サポート

この手順でうまくいかない場合は、以下の情報をスクリーンショットで共有してください：

  • Netlify の Deploys 画面（最新デプロイの時刻とファイル一覧）
  • ブラウザの開発者ツール（F12）の Console タブ
  • ブラウザの開発者ツール（F12）の Network タブで auth.js の読み込み状況
