# 🔧 データ修正機能（AI一括データ修正）検証ガイド

## 📋 目的
`admin-data-fix.html` のAI一括データ修正機能が正しく動作するかを検証します。

---

## 🎯 検証対象の機能

### 1. **統計情報表示**
- 店舗名要修正件数
- 重大度未設定件数
- 荷物情報なし件数
- カテゴリ空件数

### 2. **店舗名→住所変換** 🏪
- 店舗名を検出して住所に変換
- 元の店舗名を「エリア詳細」に移動
- ⚠️ **現在は模擬モード**（Google Places API未接続）

### 3. **重大度AI自動判定** 🔢
- 重大度が未設定のレコードを1-5で自動判定
- 状況説明から重大度を推測

### 4. **荷物情報AI補完** 📦
- 荷物情報が空のレコードを自動補完
- 状況説明から荷物情報を推測

### 5. **カテゴリ自動判定** 🚚
- 事象カテゴリ（走行中/荷役中）を自動分類
- 状況説明から判定

## 🐛 よくある問題と解決方法

### 問題1: ページが真っ白
**解決策:**
1. ブラウザで Ctrl+Shift+R（ハードリロード）
2. F12 → Console で JavaScript エラーを確認
3. Supabase クライアントが初期化されているか確認

### 問題2: 統計情報が「-」のまま
**解決策:**
1. F12 → Console で「📊 統計情報取得開始」ログを確認
2. Supabase接続エラーがないか確認
3. incidents テーブルにデータが存在するか確認

### 問題3: ボタンをクリックしても何も起こらない
**解決策:**
1. F12 → Console でエラーログを確認
2. ブラウザのキャッシュをクリア
3. JavaScriptが正しく読み込まれているか確認

### 問題4: プログレスバーが進まない
**解決策:**
1. ネットワークタブで Supabase API リクエストを確認
2. レート制限エラーがないか確認
3. Supabase の RLS ポリシーを確認
