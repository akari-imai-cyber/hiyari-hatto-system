システムの概要

このシステムでできること

✅ ヒヤリハット報告を簡単に入力

✅ 過去の報告を確認

✅ 報告内容の修正を依頼

✅ スマートフォンでも操作可能

対象者

  • ドライバー（一般ユーザー）
  • 運行業務に従事する方

ログイン方法

1️⃣ ログイン画面を開く

以下のURLにアクセスしてください：

https://stellular-profiterole-2ff0a2.netlify.app/index.html
🔗 ログイン画面を開く

2️⃣ ログイン情報を入力

項目 入力内容 例
企業コード 会社から指定されたコード nlp-test
メールアドレス あなたのメールアドレス driver1@nlp-test.com
パスワード 会社から指定されたパスワード Driver123!

3️⃣ ログインボタンをクリック

入力が完了したら、「ログイン」ボタンをクリックしてください。

ヒヤリハット報告の入力

ステップ1: 基本情報を入力

ログイン後、以下の情報を入力してください：

📝 入力項目

項目 説明 入力例
報告者名 あなたの名前 山田太郎
発生日 ヒヤリハットが起きた日 2026-03-01
発生時刻 ヒヤリハットが起きた時間 14:30
発生場所 具体的な場所 東京都千代田区丸の内1-1-1

🗺️ GPS位置情報の取得（任意）

💡 「現在地を取得」ボタンをクリックすると、自動で位置情報が入力されます

💡 ブラウザが位置情報の許可を求めた場合は、「許可」をクリックしてください

ステップ2: 詳細情報を入力

「次へ」ボタンをクリックすると、詳細入力画面が表示されます。

📋 詳細項目

項目 説明 選択肢
カテゴリ ヒヤリハットの種類 運転操作ミス / 車両不具合 / 交通環境 / その他
詳細 具体的な状況を記入 自由記述（できるだけ詳しく）

✍️ 詳細記入のポイント

良い例：

信号待ちで停車中、後方から急接近する車両を発見。 急ブレーキで回避したが、車間距離約2mまで接近された。 時刻：14:30頃、場所：国道1号線、天候：晴れ

悪い例：

危なかった

ステップ3: 報告を送信

すべての入力が完了したら、「報告を送信」ボタンをクリックしてください。

✅ 「報告が送信されました」というメッセージが表示されれば完了です！

過去の報告を確認する

📊 ダッシュボードを開く

ログイン後、画面上部のナビゲーションから 「ダッシュボード」をクリックします。

報告一覧の見方

列名 説明
報告者 報告した人の名前
発生日時 ヒヤリハットが起きた日時
場所 発生場所
カテゴリ ヒヤリハットの種類
ステータス 対応状況（未確認 / 確認済み / 対応中 / 完了）
操作 詳細表示・修正依頼

🔍 報告の詳細を見る

各報告の右側にある 「詳細」ボタンをクリックすると、報告の全内容が表示されます。

報告内容の修正を依頼する

修正が必要な場合

報告内容に誤りがあった場合や、追加情報を記載したい場合は、修正依頼を送信できます。

修正依頼の手順

  1. ダッシュボードから修正したい報告の 「詳細」をクリック
  2. 画面下部の 「修正依頼」セクションを見つける
  3. 修正内容を入力
    例：「発生時刻を14:30から15:00に修正してください」
  4. 「📤 修正依頼を送信」ボタンをクリック

✅ 企業管理者に修正依頼が通知されます

✅ 管理者が修正を完了すると、報告内容が更新されます

よくある質問（FAQ）

Q1: ログインできません

A: 以下を確認してください：

  1. 企業コードが正しいか
    • 会社から指定されたコードを入力していますか？
    • 大文字・小文字は区別されます
  2. メールアドレスが正しいか
    • スペースや誤字がないか確認してください
  3. パスワードが正しいか
    • パスワードは大文字・小文字を区別します
    • コピー＆ペーストする場合は、余分なスペースに注意
  4. ブラウザのキャッシュをクリア
    • Ctrl + Shift + Del でキャッシュを削除してください

Q2: GPS位置情報が取得できません

A: 以下を確認してください：

  1. ブラウザの位置情報許可
    • ブラウザの設定で位置情報の使用を許可してください
  2. デバイスのGPS機能
    • スマートフォンの場合、GPS機能がオンになっているか確認
  3. 手動入力も可能
    • GPS取得ができなくても、住所を手動で入力できます

Q3: 報告を送信した後に間違いに気づきました

A: 修正依頼を送信してください：

  1. ダッシュボードから該当報告を開く
  2. 「修正依頼」セクションに修正内容を記入
  3. 送信ボタンをクリック

企業管理者が確認して修正します。

Q4: パスワードを忘れました

A: 企業管理者に連絡してください：

  • 企業管理者がパスワードをリセットします
  • 新しいパスワードが通知されます

Q5: スマートフォンで使えますか？

A: はい、使えます！

  • iPhoneでもAndroidでも利用可能
  • SafariやChromeなどのブラウザで開いてください

Q6: 写真は添付できますか？

A: 現在のバージョンでは写真添付機能はありません

  • 詳細欄にできるだけ詳しく状況を記述してください
  • 必要な場合は、管理者に直接写真を送付してください

💡 トラブルシューティング

ログインできない場合

  1. ブラウザのキャッシュをクリア（Ctrl + Shift + Del）
  2. シークレットモードで試す（Ctrl + Shift + N）
  3. 別のブラウザで試す
  4. 企業管理者に連絡

画面が正しく表示されない場合

  1. ブラウザを最新版に更新
  2. ページをリロード（F5またはCtrl + R）
  3. ハードリフレッシュ（Ctrl + Shift + R）

報告が送信できない場合

  1. インターネット接続を確認
  2. 必須項目がすべて入力されているか確認
  3. ブラウザのコンソール（F12）でエラーを確認
  4. システム管理者に連絡

📌 チェックリスト

報告前に以下を確認してください：

  • 報告者名を入力した
  • 発生日時を正確に入力した
  • 発生場所を具体的に記入した
  • カテゴリを選択した
  • 詳細を記述した（できるだけ詳しく）
  • 内容を確認した
  • 送信ボタンをクリックした
