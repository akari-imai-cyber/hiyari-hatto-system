📘 操作フロー & トラブルシューティング

📝 ヒヤリハット報告の入力フロー

1
ログイン → 企業コード、メール、パスワードを入力
2
基本情報入力 → 報告者名、発生日時、場所
3
「次へ」をクリック
4
詳細入力 → カテゴリ選択、詳細記述
5
「報告を送信」をクリック → 完了！

📩 修正依頼の送信フロー

1
ダッシュボードを開く
2
修正したい報告の「詳細」をクリック
3
画面下部の「修正依頼」セクションに修正内容を記入
4
「📤 修正依頼を送信」をクリック → 完了！

👥 ユーザー追加フロー（企業管理者）

方法: Supabase経由（推奨）

1
Supabase → Authentication → Users → Add user
2
メール、パスワード入力、Auto Confirm Userをチェック
3
作成されたユーザーのUIDをコピー
4
SQL EditorでINSERT文を実行（プロフィール情報追加）
5
ユーザー管理画面で確認 → 完了！

🔧 トラブルシューティング

❌ ログインできない

  • ✓ 企業コード、メール、パスワードを再確認
  • ✓ ブラウザキャッシュをクリア（Ctrl + Shift + Del）
  • ✓ シークレットモードで試す（Ctrl + Shift + N）
  • ✓ 企業管理者に連絡

❌ リンクが表示されない（管理者機能）

  • ✓ ハードリフレッシュ（Ctrl + Shift + R）
  • ✓ ロールがcompany_adminか確認（F12コンソール）
  • ✓ 最新版がデプロイされているか確認

❌ GPS位置情報が取得できない

  • ✓ ブラウザの位置情報許可を確認
  • ✓ デバイスのGPS機能を確認
  • ✓ 手動で住所を入力（GPS不要）

❌ 画面が正しく表示されない

  • ✓ ブラウザを最新版に更新
  • ✓ ページをリロード（F5）
  • ✓ 別のブラウザで試す
